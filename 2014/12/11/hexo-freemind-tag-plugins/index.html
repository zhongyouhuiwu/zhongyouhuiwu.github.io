freemind主题的tag和plugins用法.

Freemind 主题提供了丰富的 tag plugins，以充分利用 Bootstrap 的威力。

在使用这些 tag plugins 之前，你需要先安装hexo-tag-bootstrap 。在你的 Hexo 博客目录，执行下列命令：

然后就可以像使用其他 hexo tag plugins 一样使用这些 Bootstrap tag plugins 了。
