pureftpd安装-Error

pureftpd安装-Error: Table 'admin' from database 'ftpusers' doesn't exist.

这是因为MySQL 5.1和5.5下的语句有些不同导致在MySQL 5.5下安装失败，其实Pureftpd是安装成功的，只不过php的图形界面无法登陆。

MySQL 5.5 且使用了pureftpd的用户需要按如下方法修复，执行如下命令：

按提示分别输入MySQL root密码，ftp用户管理面板密码和MySQL ftp用户密码。

然后在浏览器里输入： http://服务器域名或ip/ftp 开始设置ftp账号
